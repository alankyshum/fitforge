# .learnings/ — BLD Knowledge Base

This directory contains transferable learnings extracted from completed work by the knowledge-curator agent.

## Structure

- **INDEX.md** — Master index of all learnings
- **patterns/** — Reusable patterns that worked
- **pitfalls/** — Things that went wrong and how to avoid them
- **decisions/** — Architecture and technology decisions with rationale
- **process/** — Process improvements and workflow learnings
- **debugging/** — Debugging techniques and common error patterns

## For Agents

Before starting any task, check INDEX.md for relevant learnings. Past mistakes and discoveries are documented here to prevent repeating them.

## Maintenance

This directory is maintained by the knowledge-curator agent. Do not manually edit — submit learnings through the issue tracking system.
